## 1. 이해

    음이 아닌 정수가 주어지면 그 정수만큼 파스칼 삼각형을 완성한다.

## 2. 계획

    첫번째 줄과 두번째 줄은 정해져있는 답이고, 나머지 줄부터는 첫번째와 마지막은 1이고 그 사이값들을 계산해야한다.
    이중 for문을 사용해야 할 것 같고 if 문으로 경우의 수를 나눠서 생각해본다.
